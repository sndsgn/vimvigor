r
Replace the character under the cursor with the character you type without leaving Normal mode.
r{char}
r{char}z Replace the character under your cursor with the letter "z".
R
REPLACE mode will be entered and every character typed will replace the chracter under the cursor.
R
R Replace mode is entered and every character typed will replace the chracter under the cursor.
s
Substitute the character under the cursor with the character you type now in Insert mode. Synonym: cl.
[count]s
23s Substitute the character under the cursor and the following 22 characters with what I type.
S
SUBSTITUTE the current line and the number of lines specified with the characters you type now in Insert mode. Synonym: 0̸C.
[count]S
23S Substitute the current word under the cursor and the following 22 characters with what I type.
t
Till before the [count] instance of the character specified to the right on the current line including the character under the cursor implement the operator action specified.
[count]operatort{char}
23cta Till before the 23rd instance of the letter "a" to the right, including the character under the cursor cut all of the text to the buffer and enter insert mode.
T
Till after the [count] instance of the character specified to the left on the current line excluding the character under the cursor implement the operator action specified.
[count]operatorT{char}
23cTa Till after the 23rd instance of the letter "a" to the left excluding the character under the cursor cut all of the text to the buffer and enter insert mode.
u
Undo [count] changes throughout the entire file.
[count]u
23u Undo all 23 previous changes throughout the entire file.
U
UNDO all recent changes made on the line where the cursor is. When you use the U command more than once it will toggle between undoing and repeating changes for the line on which the cursor is.
[count]U
23U Undo all changes on the 23 previous lines on which there were changes.
v
Visual mode per character is entered selecting the character under the cursor and all characters in the direction multiple specified. Additional commands specifically {motion} to highlight and an operator to implement an action on the highlighted characters are required.
v[count]{motion}operator
gv"xc Visually select the previous Visual mode selection and cut it to register x.
V
VISUAL mode per line is entered selecting the line under the cursor and all lines in the direction multiple specified. Additional commands specifically {motion} to highlight and an operator to implement an action on the highlighted characters are required.
V[count]{motion}operator
23Vy Visual mode will select the next 23 lines. Yank the selection to the buffer.
w
Word, the beginning of the [count] special character after the current cursor position and [count] string of letters after the next blank space is where the cursor should be.
[count]w
23w 23 words or strings of letters after the current cursor position or special character is where the cursor should be.
W
WORD or the beginning of the [count] string of characters after the current cursor position and after a blank space is the where the cursor should be.
[count]w
23W 23 words or strings of letters after the current cursor position or special character is where the cursor should be.
x
x or delete [count] characters including the character under the cursor.
[count]x
23x x or delete 23 characters after and including the character under the cursor.
X
X or delete [count] characters before the cursor with the default being one.
[count]X
23X X or delete 23 characters before the current cursor position.
y
Yank or copy [count] characters with the {motion} specified to the buffer.
[count]y{motion}
23yl Yank or copy to the buffer 23 characters under and to right of the cursor on the current line.
Y
YANK or copy [count] lines both under the cursor and below the current line to the buffer.
[count]Y
23Y YANK or copy to the buffer 23 lines both under the cursor and below the current line.
z<CR>
Zero the cursor line number to the top of the current screen or [count] line number after you hit return or enter.
[count]z<CR>
23z<CR> Zero the 23rd line to the top of the screen and move the cursor to the line's first non-blank character.
ZZ
ZOOM out of the current file by first saving if there have been any modifications since the last save and then quitting.
ZZ
ZZ Zoom out of the current file by first saving if there have been any modifications since the last save and then quitting.
